Gemika::JsonLdGenerator

The Gemika::JsonLdGenerator is a Ruby class that generates JSON-LD (JSON for Linking Data) for images. JSON-LD is a method of encoding Linked Data using JSON. It is used to structure data in web pages, making it easier for search engines to understand the content.

Installation

Add this line to your application's Gemfile:

gem 'gemika-json-ld-generator'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install gemika-json-ld-generator

Usage

To use the Gemika::JsonLdGenerator , you need to call the generate class method with an array of images as the argument. Each image should be a hash with the following keys:

  • url : The URL of the image.
  • license : The URL of the image's license.
  • license_page : The URL of the page containing the image's license.
  • credit_text : The credit text for the image.
  • creator_name : The name of the image's creator.
  • copyright : The copyright information for the image.
